Создат класс банкомат, в котором изначально есть какая-то сумма

Создать метод, который выводит деньги со счета. В качестве параметра принимает `имя` человека и `сумму`, которую он хочет вывести.

Последовательность внутри метода:

Выводится строка: `Имя` подошел к банкомату

Поток останавливатьестся на 2 секунды

Осуществляется проверка - если в банкомате достаточно денег для выдачи, то выводится "`Имя` вывел `сумма` рублей. В банкомате осталось оставшаяся `сумма` рублей"

Если в банкомате недостаточно денег, то выводится строка " В банкомате недостаточно денег для `имя`.

В методе main создайте несколько потоков, каждый из которого будет представлять собой клиента, который хочет вывести деньги. Намеренно сделайте так, чтобы кому-то не хватило денег и запустите код на выполнение.

Из-за отсутствия синхронизации возникнет гонка. После того как деньги закончились, кто-то ещё смог их вывести, в банкомате осталась отрицательная сумма. Когда убедитесь что проблема есть, решите её 2 способами.
